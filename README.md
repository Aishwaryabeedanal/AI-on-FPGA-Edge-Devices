# AI-on-FPGA-Edge-Devices
# Edge AI Object Detection on Raspberry Pi and Jetson

## 🔹 Overview
This project focuses on deploying an object detection model on edge devices such as Raspberry Pi and NVIDIA Jetson for real-time inference.

## 🔹 Key Features
- Deployment of AI model on edge hardware
- Real-time object detection
- Optimized performance for embedded systems
- Cross-device implementation

## 🔹 Description
An object detection model was deployed on Raspberry Pi and NVIDIA Jetson platforms to enable real-time inference on edge devices. The system demonstrates efficient AI deployment in resource-constrained environments.

The project highlights optimization techniques for running deep learning models on embedded hardware.
